최근 업데이트: April 1, 2019

BOS Platform Foundation(이하 우리)는 www.bosneo.io 웹사이트(이하 서비스)에서 쿠키를 사용합니다. 본 서비스를 이용함으로써 사용자는 쿠키 사용에 동의하게 됩니다. 우리의 쿠키 정책은 쿠키가 무엇인지, 쿠키를 어떻게 사용하는지, 우리의 제 3자 서비스에서 쿠키를 어떻게 사용하는지, 사용자의 쿠키에 관한 선택 및 추가정보를 설명합니다.

쿠키란

쿠키는 방문하는 웹사이트에서 웹 브라우저로 보내는 작은 텍스트들 입니다. 하나의 쿠키 파일은 사용자의 웹 브라우저에 저장되고 서비스 또는 제 3자 서비스에서 재 방문을 좀더 쉽게 하거나 좀더 유용하게 서비스를 제공하는 것을 허용합니다. 쿠키는 "영구적"이거나 "세션" 것일 수 있습니다. 영구적인 쿠키는 오프라인 상태에서 사용자의 컴퓨터 또는 모바일 장치에서 유지될 수 있으나 세션 쿠키는 웹 브라우저를 닫으면 즉시 삭제됩니다.

쿠키의 사용

사용자가 서비스에 접속하고 있을 때 우리는 사용자의 웹 브라우저에 다수의 쿠키 파일을 배치 시킬 수 있습니다. 우리는 다음 목적으로 쿠키를 사용합니다.
1. 서비스의 특정 기능을 사용하기 위해
2. 분석을 하기 위해
3. 환경 설정을 저장하기 위해 우리는 서비스에서 영구적인 쿠키와 세션 쿠키를 모두 사용하고 서비스를 실행하기 위해 다양한 유형의 쿠키를 사용합니다.
1. 필수쿠키: 우리는 쿠키를 사용하여 서비스가 사용자의 언어 환경과 같은 서비스의 작동 또는 표기가 바뀌는 정보를 기억할 수 있습니다.
2. 계정쿠키: 우리는 사용자 계정의 불법적인 사용을 예방하고 사용자를 인정하기 위해 계정쿠키를 사용할 수 있습니다. 우리는 이런 쿠키를 이용하여
"remember me" 같은 기능처럼 서비스의 작동 또는 표기가 바뀌는 정보를 기억할 수 있습니다.
3. 분석쿠키: 우리는 서비스를 개선할 수 있도록 하는 정보를 추적할 수 있도록 쿠키 정보를 사용할 수 있습니다. 우리는 분석 쿠키를 사용하여 새로운광고, 페이지, 특징, 기능을 테스트하여 사용자의 반응을 확인 할 수 있습니다.

제 3자 쿠키

우리의 쿠키 외에도 서비스의 분석과 서비스를 통한 광고 전달율을 보고하기 위해 제 3자 쿠키를 사용할 수 있습니다.

쿠키의 선택에 따른

사용자가 쿠키를 삭제하거나 웹브라우저에 쿠킥 삭제 또는 거부를 지시하려면, 사용자의 웹 브라우서 도움말 페이지를 방문하세요. 쿠키를 삭제하거나 쿠키 사용에 대해 동의하지 않을 경우 우리가 제공하는 모든 기능을 사용할 수 없거나 환경설정을 저장하지 못할 수 있으면 일부 페이지는 정상적으로 표시되지 않을 수 있습니다.
1. 크롬 웹 브라우저: https://support.google.com/accounts/answer/32050
2. 익스플로어 웹 브라우저: http://support.microsoft.com/kb/278835
3. 모질라 웹 브라우저: https://support.mozilla.org/en-US/kb/delete-cookies-remove-info-websites-stored
4. 사파리 웹 브라우저: https://support.apple.com/kb/PH21411?locale=en_US
5. 기타 웹 브라우저는 해당 브라우저의 공식 웹사이트를 방문하세요.
